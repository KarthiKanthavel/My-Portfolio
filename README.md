# My Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my projects, skills, and contact information.  
Designed with a clean, elegant UI and subtle animations for a polished feel.

## Features
- Single-page layout with smooth scrolling sections
- Modern hero section with animated background effects
- Project showcase section
- About & contact sections
- Responsive design for desktop and mobile
- Clean, minimal aesthetic with glassmorphism elements

## Tech Stack
- **React** + **TypeScript**
- **Vite** (fast development & build tooling)
- **Tailwind CSS**
- **React Bits** (UI effects & components)
